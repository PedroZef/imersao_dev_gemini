# Base de Conhecimento de Linguagens de Programação

## Este projeto é uma base de conhecimento interativa que permite aos usuários pesquisar informações sobre diversas linguagens de programação. Com uma interface amigável, os usuários podem digitar o nome de uma linguagem e obter detalhes relevantes, como características, usos comuns e exemplos de código.

## Funcionalidades

-   Pesquisa interativa: Os usuários podem digitar o nome de uma linguagem de programação para obter informações detalhadas.
-   Tema claro/escuro: Um botão permite alternar entre temas claro e escuro para melhor visualização.
-   Design responsivo: A interface se adapta a diferentes tamanhos de tela, garantindo uma boa experiência em dispositivos móveis e desktops.

## Tecnologias Utilizadas

-   HTML5: Estrutura da página web.
-   CSS3: Estilização da interface, incluindo temas claro e escuro.
-   JavaScript: Funcionalidade de pesquisa e alternância de temas.

## Como Usar

1. Clone este repositório para o seu ambiente local.
2. Abra o arquivo `index.html` em um navegador web.
3. Digite o nome de uma linguagem de programação na barra de pesquisa para ver as informações correspondentes.
4. Clique no botão de tema para alternar entre os modos claro e escuro.
